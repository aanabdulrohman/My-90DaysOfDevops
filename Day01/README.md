# Day 01 - Basic Linux
- Belajar navigasi terminal (pwd, ls, cd, dll)
- Belajar manajemen file (touch, mkdir, echo, cat, dll)
- Membuat bash script sederhana

## Pembelajaran yang di dapat
- Tau bagaimana cara berpindah folder, buat folder dan buat file
- monitorig resouce device dengan menggunakan top/htop
- filter kata terntu dengan grep dan power of fipping (|)
- check disk space dan memory menggunakan df -h dan free -m
- Redirect otput menggunakan `>` (overwrite) dan `>>` (append/menambahkan)
- mengalihkan error menggunakan (`/dev/null`)
- dan yang paling penting di pelajaran kali ini, membuat system check sederhana, dari pembelajaran yang sudah di dapat
